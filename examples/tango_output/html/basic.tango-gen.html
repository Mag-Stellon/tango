About Tango
Tango is a document processing system similar to \verb|Sphinx| or Pandoc with
some
specificities.

  • its input format is a simplified variant of LateX

  • it is primarily aimed at two output formats:


    • a latex output for producing PDF files (for screen or printout)

    • a html/css/javascript format for interactive web pages


  • its main goal is for producing scientific and technical document with

    • enhanced interactivity without sacrifying the availability of a high-quality printable static version.

For very pragmatic reasons Tango is written in http://www.python.org.
